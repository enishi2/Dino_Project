# Co-op Puzzle Client

This is the web game client for the Dino Memo cooperative puzzle mode.

## Planned stack

- Phaser
- PartyKit for realtime room sync

## Local dev

```powershell
cd "C:\Naydino Project\cooppuzzle-client"
npm install
npm run dev
```

## First milestone

Render a room and connect two players to the same PartyKit room.
